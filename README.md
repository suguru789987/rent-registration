# rent-registration

ラクミー経営管理 地代家賃機能の仕様・モック・検証プラン一式。

> **制作:** [@suguru789987](https://github.com/suguru789987) (PdM)
> **担当範囲:** 業界構造モデリング(固定/変動/段階別歩合3型) / UIモック / 検証プラン18改訂 / 先出リリース+切り戻しプラン設計 / ヘルプページ差分管理
> **ツール:** Claude Code / HTML / CSS / Markdown / TSV

---

## ハイライト

- 地代家賃の**3型モデル**(固定賃料 / 変動賃料 / 段階別歩合) を設計
- **最低保証 × 対象売上率 × 適用開始日**の3属性で計算分岐
- 検証プランは**損益分岐点分析(CVP)付き計算根拠**を12列で体系化(P-0〜R-9)
- 先出リリース/フルバージョン/切り戻しの**3段階リリース設計**
- 2026/03/19〜04/08 の**3週間で検証プラン18改訂**

## ディレクトリ構成

| ディレクトリ | 内容 |
|---|---|
| `mock/` | 地代家賃モック(旧 monthly-expense-rent-mock) |
| `modal/` | 賃料モーダルモック(旧 rent-modal-mock) |
| `diff/` | 切り出し版 前回リリース差分リスト |
| `verification/` | 検証プラン・不具合一覧 |
